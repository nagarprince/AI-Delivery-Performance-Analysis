# AI-Based Delivery Performance Analysis Dashboard

## Project Overview

This project focuses on analyzing delivery operations and identifying factors that contribute to delivery delays. The solution combines Machine Learning techniques with an interactive Power BI dashboard to provide business insights and improve delivery performance.

The dashboard enables users to monitor key delivery metrics, analyze delay patterns, and make data-driven decisions.

---

## Project Objectives

* Analyze delivery performance data
* Identify factors affecting delivery delays
* Build a Machine Learning model to predict delivery delays
* Create an interactive Power BI dashboard
* Generate actionable business insights

## Key Business Insights

* Stormy weather contributes to the highest delivery delays.
* Express delivery mode experiences higher average delays.
* Certain delivery partners have significantly higher delayed deliveries.
* Delay rate is approximately 21%.
* On-time delivery rate is approximately 79%.
* Regional performance analysis helps identify operational bottlenecks.
